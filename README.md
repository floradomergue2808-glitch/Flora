
//////////////////////////
/// Importer le projet ///
//////////////////////////

1- Crée un compte Git Hub.
2- Sur ce compte, creer un nouveau repertoire.
    • Soit en cliquant sur le bouton vert ( un livre avec marqué New ) en haut à gauche.
    • Soit en cliquant sur les trois trait en haut à gauche, puis Repositories.
    puis sur le bouton en haut à droite en vert New repository.
3- Dans le texte en haut cliquer sur le lien en bleu Import a repository.
4- 
    • Dans le lien pour l'URL mettre https://github.com/Titouan974/PWA_Frise-2.git .
    • Choisir un nom.
    • Laissé en Public.
    • Cliquer sur Begin import.

////////////////////////////
/// Configurer le projet ///
////////////////////////////

1- Cliquer sur les trois trait en haut à gauche, puis Repositories.
2- Touver le réportoire précédament importé puis cliquer sur le nom du repertoire.
3- Dans les onglets en haut choisir Settings.
4- Aller dans l'onglet Pages de Code ans automation.
5- 
    • Dans Build and deployment choisit Deploy from a branch.
    • En dessus choisir main.
    • Normalement juste à coté aparaît /(root) sinon le sélectionner.
    • Cliquer sur Save.
6- Attendre.
    • Pour savoir si c'est bon appuyer sur le nom du repertoire .en haut à gauche dans la fenêtre qui contient les fichiers au dessus du du README il y a votre nom, le nom du dernier commit puis un symbole.
    • Si c'est une boule orange alors il faut attendre.
    • Si c'est une coche verte alors c'est bon.

////////////////////////////
/// Ajouter des éléments ///
////////////////////////////

1- Ouvrir le fichier ./Scripts/envents.json .
2- Cliquer sur l'icon du stylo Edit this file.
3- Rajouter les nouveaux éléments en suivant la syntaxe des deux exemples ( \t est pour le retour à la ligne et \t pour une tabulation ).
4- Cliquer sur le bouton vert en haut à droite.
5- Mettre un nom au commit.
6- Cliquer sur Commit changes


///////////////////////////////////////
/// Changer le nom de l'application ///
///////////////////////////////////////

1- Ouvrir le fichier manifest.json
2- Cliquer sur l'icon du stylo Edit this file.
3- Changer les noms marqués après name et short_name.
4- Cliquer sur le bouton vert en haut à droite.
5- Mettre un nom au commit.
6- Cliquer sur Commit changes



//////////////////////////
/// Changer les images ///
//////////////////////////

1- Avoir des images en png.
2- Avoir une image de 192 * 192px et une autre de 512 * 512px.
3- Les renommer respectivement icon-192.png et icon-512.png.
4- Cliquer sur l'icon plus en haut à droite puis sur Upload files.
5- Glisser-déposer les images à l'endroit prévu pour.
6- Mettre un nom au commit.
7- Cliquer sur Commit changes
