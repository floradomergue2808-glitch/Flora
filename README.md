

///////////////////////////////////////
/// Changer le nom de l'application ///
///////////////////////////////////////

1- Ouvrir le fichier manifest.json
2- Changer les noms marqués après name et short_name.
3- Sauvegarder le fichier



//////////////////////////
/// Changer les images ///
//////////////////////////

1- Avoir des images en png.
2- Avoir une image de 192 * 192px et une autre de 512 * 512px.
3- Les renommer respectivement icon-192.png et icon-512.png.
4- Les enregistrer dans le dossier Icon.
